## JAVA

## 📌 About
This repository is created to learn and practice **Java programming** from basic to advanced concepts.  
It serves as a structured guide for understanding Java with examples and problem-solving.

---

## 🎯 Learning Goals
- Understand Java syntax and structure  
- Learn Object-Oriented Programming (OOP) concepts  
- Develop problem-solving skills  
- Prepare for exams and coding interviews  

---

## 📚 Topics Covered

### 🔹 Basics
- Introduction to Java  
- Features of Java  
- Data Types & Variables  
- Operators  
- Input/Output (Scanner, System.out)  
- Control Statements (if, switch, loops)  

### 🔹 OOP Concepts
- Classes and Objects  
- Constructors  
- Inheritance  
- Polymorphism  
- Abstraction  
- Encapsulation  

### 🔹 Advanced Topics
- Arrays & Strings  
- Exception Handling  
- File Handling  
- Multithreading  
- Collections Framework
